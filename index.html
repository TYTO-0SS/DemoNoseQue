Instrucciones para GitHub Pages

Para ejecutar esta aplicación en GitHub Pages, sigue estos pasos:

  1. Crea un nuevo repositorio en GitHub
  2. Sube este archivo HTML con el nombre index.html
  3. Asegúrate de tener un archivo targets.mind en el mismo directorio
  4. Ve a la configuración del repositorio y activa GitHub Pages en la sección correspondiente
  5. Selecciona la rama principal (main o master) como fuente
  6. Tu aplicación estará disponible en: https://[tu-usuario].github.io/[nombre-repositorio]

Notas importantes:

  • GitHub Pages requiere que el archivo se llame index.html
  • Asegúrate de que el archivo targets.mind esté en el mismo directorio
  • La aplicación necesitará permisos de cámara, así que los usuarios deberán permitir el acceso
  • Para mejores resultados, accede desde un dispositivo móvil con cámara
